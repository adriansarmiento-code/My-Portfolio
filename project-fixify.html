Overview

Fixify was built to solve a real, local problem: homeowners in Angeles City struggling to find trusted, verified professionals for essential home services. The platform creates a centralized solution to browse, book, and review service providers.

The Problem

Local homeowners relied on word-of-mouth or unverified listings — leading to wasted time and inconsistent service quality. There was no structured platform dedicated to local home services.

My Role

Designed and developed the full application — database schema, REST API, Vue.js component architecture, JWT authentication, role-based routing, and real-time notification logic end-to-end.

Key Features

01
Service Discovery

Browse verified providers across Plumbing, Electrical, and Cleaning with ratings and reviews.

02
Booking System

Customers can schedule and manage appointments with service providers in real time.

03
Real-Time Notifications

In-app alerts for booking status changes — keeping both parties informed.

04
JWT + Role-Based Auth

Secure authentication with distinct roles: customer, provider, and admin.

05
RESTful API

Well-structured API with proper HTTP methods, status codes, and error handling.

06
Responsive UI

Fully responsive Vue.js interface — seamless across mobile, tablet, and desktop.

Overview

Fixify was built to solve a real, local problem: homeowners in Angeles City struggling to find trusted, verified professionals for essential home services. The platform creates a centralized solution to browse, book, and review service providers.

The Problem

Local homeowners relied entirely on word-of-mouth or unverified online listings — leading to wasted time, inconsistent service quality, and potential scams. There was no structured, trustworthy platform dedicated to local home services.

Key Features

01
Service Discovery

Browse verified providers across Plumbing, Electrical, and Cleaning with ratings and reviews.

02
Booking System

Customers can schedule and manage appointments with preferred service providers.

03
Real-Time Notifications

In-app alerts for booking status changes and messaging updates in real time.

04
JWT Authentication

Secure role-based access control across customer, provider, and admin roles.

05
RESTful API Design

Well-structured API with proper HTTP methods, status codes, and error handling.

06
Responsive Interface

Fully responsive design built for seamless use across mobile, tablet, and desktop.

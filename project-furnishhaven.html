Overview

FurnishHaven is a full-featured PHP e-commerce web application designed to connect customers with furniture retailers. The platform provides a seamless shopping experience with product browsing, cart management, and a clean checkout flow — all implemented using server-side PHP with a MySQL product catalog.

Technical Approach

Built using PHP for server-side logic and session management, MySQL for the product and order database, and vanilla JavaScript for interactive frontend behaviors. The project demonstrates solid understanding of web fundamentals, MVC principles, and secure form handling without relying on a full framework.

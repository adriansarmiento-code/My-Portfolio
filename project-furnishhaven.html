Project Overview

FurnishHaven is a full-featured e-commerce web application built to connect customers with furniture retailers. The platform was designed to make furniture shopping more accessible, transparent, and enjoyable through a clean, intuitive interface.

Core Purpose

The platform addresses the fragmented furniture retail experience in the Philippines, where customers often must visit multiple stores to compare options. FurnishHaven centralizes product browsing, with competitive pricing and detailed product information all in one place.

Technical Implementation

Built using PHP for server-side logic with a MySQL-backed product catalog, combined with vanilla JavaScript for interactive frontend behaviors and CSS for responsive, polished styling. The project demonstrates solid MVC principles and PHP session management for cart and user state.

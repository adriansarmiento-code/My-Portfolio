Overview

FurnishHaven is a full-featured PHP e-commerce application connecting customers with furniture retailers. It provides a seamless shopping experience — product browsing, cart management, and a clean checkout flow with a relational MySQL product catalog.

The Problem

The project demonstrates deep command of server-side web fundamentals: session management, SQL queries, secure form handling, and dynamic page generation — the foundations every modern full-stack application is built on.

My Role

Designed and built the full application using PHP for server-side logic, MySQL for the relational database, and vanilla JavaScript for interactive front-end behaviors including cart updates and form validation.

Key Features

01
Product Catalog

Browsable catalog with categories, search, and individual product detail pages.

02
Cart Management

Session-based shopping cart — add, update quantity, and remove items before checkout.

03
User Authentication

Secure login and registration with password hashing and PHP session management.

04
Checkout Flow

Clean multi-step checkout process with order summary and confirmation.

05
Responsive Design

Fully adaptive layout across mobile, tablet, and desktop — no frameworks, pure CSS.

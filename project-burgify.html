Overview

Burgify was one of Adrian's foundational projects — built before any framework knowledge. The focus was on mastering the building blocks of the web: semantic HTML structure, CSS layout systems, visual hierarchy, and responsive design. Every pixel written by hand.

The Problem

Understanding the fundamentals deeply separates strong developers from those who only know frameworks. This project proves that before reaching for Vue or React, there was solid mastery of the raw tools the web is built on.

My Role

Solo project — all HTML structure, CSS styling, layout systems, and responsive breakpoints written from scratch with zero external dependencies.

Key Features

01
Semantic HTML5

Proper use of landmark elements, heading hierarchy, and accessible markup throughout.

02
CSS Grid & Flexbox

Complex multi-column product grids and flexible nav/hero layouts — no helper library.

03
Responsive Design

Fully adaptive layouts across mobile, tablet, and desktop — breakpoints written by hand.

04
CSS Custom Properties

Design token system for colors, spacing, and typography — consistent, maintainable styles.

05
Visual Design

Deliberate color palette, typographic hierarchy, and component styling — all hand-crafted.

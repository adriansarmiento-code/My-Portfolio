Overview

Burgify was one of the foundational projects in Adrian's development journey, built before any framework knowledge. The focus was on mastering the building blocks of the web — semantic HTML structure, CSS layout systems, visual hierarchy, and responsive design principles. Every pixel was written by hand.

What It Demonstrates

Strong command of CSS Flexbox and Grid for complex page layouts, CSS custom properties for design system consistency, semantic HTML for accessibility and structure, and responsive design without a CSS framework. These fundamentals underpin every subsequent project in the portfolio.
